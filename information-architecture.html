Value Proposition

My expertise in Information Architecture (IA) focuses on organizing and structuring content for usability and discoverability, ensuring that users can quickly find the information they need.

  • Content Structuring: Designed hierarchical structures for documentation, enhancing navigation and user experience.
  • Taxonomy & Metadata: Developed taxonomy frameworks to categorize and tag content, improving searchability.
  • User-Centered Design: Collaborated with UX teams to create intuitive navigation, aligning content layout with user journeys.

Tools & Technologies

  • Content Management: WordPress, Drupal
  • User Testing: Hotjar, UserTesting
  • Collaboration Platforms: Confluence, Miro
